Run the application as jar from command line f.e. with:

java -jar act-calc-fnc.jar --dsf 2002-01 --dst 2021-09 --drf 2021-09 --fos stock.csv --too hgb

The application will create two subdirectories: "./res" and "./wrk".
The WORKING Files located in "./wrk".
The RESULTS are stored in the application directory: "./res".

See the Dockerfile for details
